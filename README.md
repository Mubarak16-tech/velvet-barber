# The Velvet Barber Website

A premium, responsive barber shop website built with HTML5, CSS3, and vanilla JavaScript.

## Features
- Responsive multi-page layout
- Modern luxury styling with glassmorphism and gold accents
- Mobile-friendly navigation
- Scroll animations and reveal effects
- Testimonial slider
- Gallery filtering and lightbox
- Booking form with client-side validation
- FAQ accordion and back-to-top button

## Structure
- index.html
- about.html
- services.html
- gallery.html
- booking.html
- contact.html
- css/style.css
- js/script.js
- images/

## Run Locally
Open any HTML file in a browser, or serve the folder with a simple local server such as:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000/
